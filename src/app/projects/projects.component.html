Ghost Writers

Ghost Writers uses Wikitude's image recognition software to create a mobile augmented reality graffiti application that allows users to post and create their own virtual art, anywhere in the world. TechStack: Angular 2, Ionic 2, Wikitude, bCrypt, AWS, MongoDB, Node, Express, Google Maps API, FileStack.

Triplit

Triplit is a web application that makes it easy for you and your friends to plan trips in as simple as three easy steps. Create a trip, invite your friends, find a hotel, and depart on your adventure. TechStack: React, MySQL, Sequelize, Node, Express, Auth0, GoogleMaps API.

Slack Overflow

Slack Overflow is a web application where software engineers can go to post questions and discuss about topics based on programming, coding, technologies, and software engineering. My team had to add features to their existing codebase. I created a simple video-chatting system built on Web RTC, HTML5 and Socket.io. TechStack: React, WebRTC, Socket.io, Heroku.

Spaced Bar

A simple game. How fast can you press the space bar before the timer runs out? Created on Angular.JS, Persisted Highscore with MySQL. TechStack: Angular 1.6, MySql, Sequelize, Node, Express.

Coffee-Coffee

A simple mobile coffee app made on Ionic 2. Choose the amount of coffee with the slider. Choose the amount of milk to add on top, but add too much milk and the cup will overflow. Created all the artwork and animations myself. Animations created on Harmony Toon Boom.
